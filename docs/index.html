Shell-First Demo

ww Demo

A recorded terminal session showing the built-in interactive selector, path switching, and `ww new` flow inside a throwaway Git repository. This page exists because GitHub README rendering strips playable terminal media.

What This Covers

  1. Open `ww` and drive the built-in selector with arrow keys.
  2. Confirm the shell has switched into an existing worktree.
  3. Create a fresh worktree with `ww new feat-demo`.
  4. List worktrees again from the newly created branch.
